## NextJs — Tracking Transactions App

Run the development server:

```bash
pnpm dev
```
---
## Tech Stack

### Core
- Next.js
- React
- TypeScript
- Tailwind CSS

### Forms & Validation
- react-hook-form — form management
- zod — schema validation
- @hookform/resolvers — integration RHF + Zod

### UI & UX
- lucide-react — icons
- @tailwindcss/forms — better form styles
- react-error-boundary — error handling

### Backend & Auth
- @supabase/supabase-js — database & API
- @supabase/ssr — SSR authentication with cookies

### Utilities
- react-cookie (js-cookie is better)
